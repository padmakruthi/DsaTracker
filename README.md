
# DSA Mentor – 70 Day Learning Plan

DSA Mentor is a structured **70-day Data Structures & Algorithms learning tracker** designed to help students systematically learn and practice DSA concepts.

The application provides a **daily study plan, concept checklist, coding problems, revision schedule, and learning resources** to help beginners progress from fundamentals to advanced topics.

---

## 🚀 Features

* 📅 **70 Day DSA Learning Roadmap**
* 📚 Topic-wise learning (Arrays, Strings, Linked Lists, Trees, Graphs, etc.)
* 🧠 Concept checklist for every day
* 💻 Curated coding problems from:

  * LeetCode
  * GeeksforGeeks
* 🔁 Revision tasks to reinforce learning
* ⚠️ Common mistakes and tips
* 🎥 Learning resources (videos & articles)

## 📖 Learning Plan Topics

The 70-day roadmap covers:

* Arrays & Prefix Sum
* Two Pointers & Sliding Window
* Strings & Hashing
* Linked Lists
* Stack & Queue
* Binary Search
* Recursion & Backtracking
* Trees & Binary Search Trees
* Heap / Priority Queue
* Greedy Algorithms
* Graphs
* Advanced Problem Solving
